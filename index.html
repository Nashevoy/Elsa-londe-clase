Mis Proyectos

Optimización de Redes Empresariales

Diseño y optimización de la red interna para una empresa mediana con 200 empleados.

Optimización de Redes Empresariales
Este proyecto consistió en analizar y mejorar el rendimiento de la red interna de una empresa. Se implementaron nuevas configuraciones de switches y routers, asegurando mayor velocidad, seguridad, y redundancia en el sistema de red para reducir el tiempo de inactividad.
Implementación de VLANs en Entorno Corporativo

Segmentación de la red para mejorar la seguridad y la eficiencia.

Implementación de VLANs en Entorno Corporativo
Este proyecto trató la implementación de VLANs para una empresa con múltiples departamentos. Se dividió la red física en redes virtuales independientes para mejorar la seguridad, reducir el tráfico no deseado y mejorar la eficiencia en la transmisión de datos.
Configuración de Servidores DNS

Implementación de servidores DNS internos para una infraestructura de red.

Configuración de Servidores DNS
Configuración y despliegue de servidores DNS para gestionar la resolución de nombres dentro de una infraestructura de red local. Se configuraron tanto servidores primarios como secundarios para garantizar la disponibilidad y redundancia.
Implementación de Redes Inalámbricas Seguras

Diseño e instalación de redes Wi-Fi seguras en entornos corporativos.

Implementación de Redes Inalámbricas Seguras
En este proyecto se implementó una red inalámbrica segura en una empresa, utilizando tecnologías como WPA3 y control de acceso basado en autenticación 802.1X. Se garantizó el acceso controlado y seguro a los recursos internos de la empresa.
Monitorización de Redes con Zabbix

Monitorización de Redes con Zabbix
Este proyecto implementó una plataforma de monitorización en tiempo real utilizando Zabbix. El sistema permite detectar fallos, congestiones y comportamientos anómalos en la red, lo que facilita una respuesta proactiva a problemas antes de que afecten a los usuarios.
Configuración de Firewall Avanzado para Data Center

Implementación y configuración de un firewall avanzado para proteger el tráfico de un Data Center.

Configuración de Firewall Avanzado para Data Center
En este proyecto se implementó y configuró un firewall avanzado para un centro de datos, utilizando políticas estrictas de control de tráfico. Se establecieron reglas de filtrado para proteger los servidores internos frente a amenazas externas y controlar el acceso a los recursos según la procedencia del tráfico.
